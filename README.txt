Reference Configurator Wordpress plugin

Description:
1. Activate plugin
2. Fill a data in 'projects' custom posts in Wordpress dashboard.
3. Use a Wordpress shortcode: [references] to display and manage projects.
3. Drag and drop projects to droppable area and generate pdf references.

Instalation:
1. Upload `reference-configurator` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
