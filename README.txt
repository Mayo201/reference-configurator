Reference Configurator Wordpress plugin

Drag and drop projects and generate pdf references.

Instalation:
1. Upload files to `reference-configurator` directory and copy it to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Usage:
1. Fill a data in 'projects' custom posts in Wordpress dashboard.
2. Use a Wordpress shortcode: [references] to display and manage projects.
3. Drag and drop projects to droppable area and generate pdf references.
